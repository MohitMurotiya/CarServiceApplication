## Run Locally

- Clone the project

```bash
  git clone https://github.com/MohitMurotiya/CarServiceApplication.git
```

- Go to the project directory

```bash
  cd CarServiceApplication
```

- Install dependencies

```bash
  npm install
```

- Start the server on http://localhost:3000/

```bash
  npm start
```
